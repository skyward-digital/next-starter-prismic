# Footer

The `Footer` is a global component that is used on every page. It is added using the [Layout]() component and passed props from there

It contains an optional Call To Action, similar to the [Call To Action](?path=/docs/components-calltoaction--default) component

</Preview>

## Footer without CTA

It's also possible to display the footer without the Call To Action if needed

This can be done by omitting the `ctaTitle` and `ctaDescription` props
